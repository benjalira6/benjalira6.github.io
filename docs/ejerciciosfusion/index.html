2.2 - Ejercicios Fusion 360

Ejercicio 1 - Plato

  • Crear y dibujar sketch.

  • Revolve para dar forma y volumen al sketch.

  • Chamfler para modificar bordes.

  • Section analysis para revisar cortes.

Renders

Ejercicio 2 - Disco + hendidura

  • Crear sketch y dibujar circunferencia + líneas.

  • Intersect entre cruvas.

  • Sweep para dar forma y volumen.

  • Crear nueva circunferencia y extruirla.

  • Combinación y corte de la circunferencia extruida.

  • Section analysis para revisar cortes.

  • Accesibility para comprobar su desmoldura (en este caso, los sectores rojos muestran un restringimiento).

Renders

Ejercicio 3 - Molde

  • Create form + Plane.

  • Modify para modificar en cualquier dirección las líneas.

  • Crear rectángulo en sketch para luego extruirlo.

  • Thicken form para darle volumen a la forma y luego cortarlo.

  • Nuevo rectángulo de mayor medida para formar molde.

  • Nuevo sketch para realizar la forma de la manilla.

  • Extrude y Fillet para dar volumen a la forma y pulir bordes.

  • Press pull para quitar volumen y ajustar moldaje.
