Circuitos con LEDs: Blink

Este circuito simple consiste en un código para encender la LED durante un periodo de tiempo. En este caso se encenderá en 1 segundo y se apagará en 1 segundo, repitiendo el proceso.

Componentes

  • Placa Arduino
  • Resistencia
  • LED

Código

void setup() { pinMode(13, OUTPUT);

}

void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000);

}

Potenciómetro

Este ejercicio consiste en el encendido del LED mediante un potenciómetro, regulando la corriente.

Componentes

  • Placa Arduino
  • Resistencia
  • LED
  • Potenciómetro

Código

int sensorPin = A0; int ledPin = 13;

void setup() { // put your setup code here, to run once: pinMode(ledPin, OUTPUT); }

void loop() { int sensorValue; sensorValue = analogRead(sensorPin); digitalWrite(ledPin, HIGH); delay(sensorValue); digitalWrite(ledPin, LOW); delay(sensorValue); }

LED RGB

Este ejercicio consiste en un código que ordenará al LED RGB a encenderse en diferentes momentos y en diferentes colores.

Componentes

  • Placa Arduino
  • 3 Resistencias
  • LED RGB

Código

const int RED_PIN = 9; const int GREEN_PIN = 10; const int BLUE_PIN = 11;

int DISPLAY_TIME = 100; //In miliseconds

void setup() { pinMode(RED_PIN, OUTPUT); pinMode(GREEN_PIN, OUTPUT); pinMode(BLUE_PIN, OUTPUT);

}

void loop() { digitalWrite(RED_PIN, HIGH); digitalWrite(GREEN_PIN, LOW); digitalWrite(BLUE_PIN, LOW); delay(1000);

digitalWrite(RED_PIN, LOW); digitalWrite(GREEN_PIN, HIGH); digitalWrite(BLUE_PIN, LOW); delay(1000);

digitalWrite(RED_PIN, LOW); digitalWrite(GREEN_PIN, LOW); digitalWrite(BLUE_PIN, HIGH); delay(1000);

}

Múltiples LEDs

El siguiente código consiste en el encendido de múltiples luces LED en un determinado tiempo.

Componentes

  • Placa Arduino
  • 8 Resistencias
  • 8 luces LED

Código

int ledPins[] = {2,3,4,5,6,7,8,9};

void setup() { int index; for(index = 0; index <= 7; index++) { pinMode(ledPins[index], OUTPUT); } }

void loop() { int index; int delayTime = 100; // milliseconds to pause

for(index = 0;index <= 7; index++) { digitalWrite(ledPins[index], HIGH); // turn LED on delay(delayTime); digitalWrite(ledPins[index], LOW); // turn LED off }

}
